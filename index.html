03

Инструкция по
установке ПО

1 // Установите удаленно водительский терминал на Android-телефон

  • Откройте на телефоне браузер, в адресной строке введите: http://nsk1.taxibravo.ru/ta1.apk
  • Дождитесь окончания загрузки, запустите файл и следуйте инструкциям по установке

2 // Подключите водительский терминал:

  • Запустите программу, нажмите значок настроек (в виде шестерёнки), введите позывной
  • Нажмите кнопку "настройки подключения", введите пароль "00100"
  • В строке "сервер1" наберите nsk1.taxibravo.ru
  • В строке "сервер2" наберите nsk2.taxibravo.ru
  • Свяжитесь со службой по работе с персоналом
  • Назовите позывной, номер машины, фамилию, последние цифры номера телефона для связи с клиентом, попросите произвести авторизацию
